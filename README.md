# AI Medical Assistant using LLM

## Overview
This project uses a Large Language Model to provide medical suggestions based on user symptoms.

## Features
- Symptom-based query system
- AI-generated medical responses
- Interactive terminal application

## Tech Stack
- Python
- OpenAI API
- NLP / LLM

## How to Run

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Run the application:
```
python app.py
```

## Sample Output

```
Enter your symptoms: fever, cough

AI Suggestion:
Based on your symptoms, this may indicate a viral infection. Please consult a healthcare professional for accurate diagnosis.
```
